# 代码块

## 方式一

用两个\`\`\`包裹起来的内容，可以显示为代码块

### 基础

- 示例效果

  ```
  123
  123
  123
  ```

- 示例语法

      ```
      123
      123
      123
      ```

### 进阶

可以为代码块注明语言类型

- 示例效果

  ```python
  def add_two_number(a, b):
      return a + b
  
  n = add_two_number(2,5)
  print(n)
  ```

- 示例语法

      ```python
      def add_two_number(a, b):
          return a + b
        
      n = add_two_number(2,5)
      print(n)
      ```

- 格式说明

      ```编程语言
      代码片段
      ```

## 方式二

在行首添加4个空格或者1个tab，也可以实现多行代码块。

建议使用**空格**，因为tab有时候不够精确。

- 示例效果

      123
      123
      123

- 示例语法

  ```
      123
      123
      123
  ```
